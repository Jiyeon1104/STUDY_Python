파이썬이란?

파이썬은 1991년 네덜란드의 수학자 '귀도 반 로섬'에 의해 개발된 프로그래밍 언어로 '비단뱀'에서 유래된 로고를 볼 수 있다.

1)인터프리터 언어이다.
- 컴파일러 언어와의 차이점 : 인터프리터는 한 줄씩 실행, 컴파일러는 덩어리로 실행
2) 플랫폼에 독립적이다. 어떤 운영체제에서도 같은 파이썬 코드를 실행할 수 있음
 - 다양한 OS (Windows, osx, rinux) 등등에서 동일한 코드로 동일한 파이썬 코드 실행 가능
3) 객체지향 언어이다.
- 클래스와 객체를 기반으로 하는 언어로 정보은닉, 상속 등의 개념을 지원함
4) 동적타이핑 언어이다.
프로그램 실행 중에 다른 작업을 처리할 수 있다.
5) 개발 속도가 빠르다.
이미 개발된 많은 다양한 라이브러리, 데이터 분석 Pandas, 수치연산 numpy, 시각화 matplotlib 등등을 사용하여 빠르게 프로그램을 작성 가능하다

*단점 : 하드웨어, 앱개발(아이폰, 안드로이드)에는 적용하기 어렵다.

###########
주석 comment
###########

주석은 프로그램 소스 코드에 설명을 작성한 것을 의미함
주석은 프로그램 실행에 전혀 영향을 끼치지 않음
유지 보수나 팀작업을 할 때 모두가 이해하기 쉽고 의미 전달이 잘 되는 주석을 작성하는 것이 바람직함

파이썬에서 주석은 #으로 시작하거나 ''' - ''' 삼중 따옴표로 표현함

###########
변수 Variable
###########

변수는 어떤 데이터를 저장할 때 사용하는 메모리 저장소(?)이다.
변수를 만들 때는 = 등호 왼쪽에 변수 명을 작성하고 등호 오른쪽에는 저장할 값을 작성함

변수 명명 규칙
1) 영문, 한글, 숫자, 밑줄로 구성됨 (그러나 보통은 변수명으로 한글을 쓰지 않음)
2) 특수문자는 사용할 수 없음 (@, !, $, % 등등)
3) 대문자와 소문자를 구분함 
4) 변수명의 첫 글자는 숫자를 사용할 수 없음
5) 예약어(print, list, dict, if, for 등등) 는 사용 불가

명명 시 권고 사항
가급적이면 소문자만 사용
변수명으로 한글은 사용하지 않음
의미있는 변수명을 사용
